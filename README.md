Sistema de Gerenciamento de Biblioteca em C
Sistema em C para controle de livros, usuários e empréstimos com persistência em arquivos, relatórios e buscas avançadas. Projeto desenvolvido inicialmente como atividade acadêmica e posteriormente expandido e refinado como projeto pessoal para portfólio.

Projeto desenvolvido em linguagem C com o objetivo de simular um sistema completo de controle de biblioteca, incluindo cadastro de livros, usuários, controle de empréstimos, devoluções, relatórios e persistência de dados em arquivos.

Funcionalidades: Cadastro de livros e usuários; Controle de empréstimos e devoluções; Cálculo automático de prazo de entrega; Verificação de atrasos;

Relatórios: Empréstimos ativos; Empréstimos em atraso; Livros mais emprestados; Busca avançada de livros por múltiplos critérios; Persistência em arquivos .txt; Sistema de backup automático;

Tecnologias utilizadas: Linguagem C; Manipulação de arquivos; Estruturas de dados; Biblioteca time.h para controle de datas; Programação estruturada.

EN

Library Management System in C

Library Management System in C for managing books, users, and loans with file persistence, reports, and advanced searches. Initially developed as an academic project and later expanded and refined as a personal project for a portfolio.

The project is developed in the C programming language with the goal of simulating a complete library management system, including book and user registration, loan and return control, reports, and data persistence in files.

Features: Book and user registration; Loan and return management; Automatic due date calculation; Late return verification.

Reports: Active loans; Overdue loans; Most borrowed books; Advanced book search by multiple criteria; Data persistence in .txt files; Automatic backup system.

Technologies used: C programming language; File handling; Data structures; time.h library for date control; Structured programming.
